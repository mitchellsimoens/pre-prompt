## Install

```sh
npm install --save-exact @mitchellsimoens/pre-prompt
```

## Usage

```json
{
  "scripts": {
    "predeploy": "pre-prompt",
    "deploy": "...."
  }
}
```

When `npm run deploy` is run, the user will be asked to proceed. The user can type any of the following choices
(case-insenstive):

- `ok`
- `sure`
- `yep`
- `yes`
